# Request Header Parser Microservice

### GET request returns JSON object about the clients IP address, language preferences and software stack.

### Live service can be accesssed here: https://vystartasv-request-header-parser.glitch.me/api/whoami

### Example output bellow:

```json
{"ipaddress":"89.212.68.15","language":"en-GB,en;q=0.9,it-IT;q=0.8,it;q=0.7,lt-LT;q=0.6,lt;q=0.5,en-US;q=0.4","software":"Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/72.0.3626.81 Safari/537.36"}
```
